# Book Digest

A simple SwiftUI iOS app for generating audio-ready digests of productivity and self-help books.

The app includes:

- A visual book picker with stylized cover art.
- OpenAI Responses API integration with a configurable model name, defaulting to `gpt-5.5`.
- Settings for a local API key.
- OpenAI text-to-speech playback with configurable speech model and voice, plus pause, resume, and stop controls.

Open `BookDigest.xcodeproj` in Xcode or build the `BookDigest` scheme for an iOS simulator.
